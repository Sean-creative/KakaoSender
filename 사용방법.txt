====================================
  카카오톡 자동 전송기 사용 방법
  (v1.0.13 기준)
====================================

▶ 실행 방법:
  1. '카카오톡전송기.command' 파일을 더블클릭하세요.
  2. "확인되지 않은 개발자" 경고가 뜨면:
     - 파일을 우클릭(Control+클릭) > "열기" 선택
     또는
     - 시스템 설정 > 개인정보 보호 및 보안 > "그래도 열기" 클릭
  3. 웹 브라우저가 자동으로 열립니다.

▶ 최초 실행 시:
  - 필요한 패키지가 자동으로 설치됩니다 (3-5분 소요)
  - 인터넷 연결이 필요합니다

▶ 권한 설정 (최초 1회):
  프로그램 실행 시 아래 권한을 허용해주세요:
  
  1. 접근성 권한:
     시스템 설정 > 개인정보 보호 및 보안 > 접근성
     → "터미널" 또는 사용 중인 터미널 앱 허용
  
  2. 화면 녹화 권한 (OCR 폴백 검증용):
     시스템 설정 > 개인정보 보호 및 보안 > 화면 녹화
     → "터미널" 또는 사용 중인 터미널 앱 허용
     ※ 접근성 API가 정상 동작하면 화면 녹화는 거의 사용되지 않습니다.
        단, 접근성 API 실패 시 자동으로 OCR로 대체되므로 허용해두는 것을 권장합니다.

▶ 사용 방법:
  1. 웹 브라우저에서 "엑셀 파일을 선택하세요" 클릭
  2. 회원 목록이 담긴 .xlsx 파일 선택
  3. "카카오톡 전송 시작" 버튼 클릭
  4. 카카오톡이 자동으로 열리며 메시지 전송이 시작됩니다

▶ 엑셀 파일 형식:
  다음 컬럼이 필요합니다:
  - 이름: 카카오톡 친구 이름과 동일해야 함
  - 등록형태: 이월 / 재등록 / 신규
  - 연령: 20대 / 30대 / 40대 등

▶ 필터링 조건:
  - 등록형태: 이월, 재등록, 신규
  - 연령: 20대, 30대
  위 조건에 해당하는 회원에게만 메시지가 전송됩니다.

▶ 친구 검증 기능 (접근성 API 우선):
  - 메시지 전송 전 카카오톡에서 친구 존재 여부를 자동으로 확인합니다
  - 친구가 없으면 해당 대상자는 건너뜁니다
  - 검증 방식:
    1순위) 접근성(AX) API — macOS가 직접 UI 텍스트를 읽어 정확도가 높습니다
    2순위) OCR(화면 인식) — AX 실패 시 자동으로 대체됩니다
  - 이모티콘·특수기호가 포함된 친구 이름도 접근성 API로 정확히 처리됩니다

▶ 모의 전송(Dry-Run) 모드:
  - 실제 메시지를 보내지 않고 전송 과정을 시뮬레이션하는 테스트 기능입니다
  - 웹 화면에서 "모의 전송 (테스트)" 체크박스를 활성화하면 동작합니다
  - 친구 검색 및 검증까지는 동일하게 수행하고,
    채팅방을 1초간 열었다가 닫는 것으로 실제 전송을 대체합니다
  - 새로운 수신자 목록을 처음 등록할 때 사전 검증 용도로 활용할 수 있습니다

▶ 동명이인 처리:
  - 전송 대상에 같은 이름이 2명 이상 있으면 전송이 차단됩니다
  - 엑셀에서 해당 이름을 구분(예: 닉네임 추가)한 후 다시 시도해주세요

▶ 보편적 클립보드(클립보드 공유) 끄기 [중요]:
  - 무엇이 문제인가:
    이 프로그램은 메시지와 이름을 '복사 → 붙여넣기'(클립보드)로 처리합니다.
    여러 애플 기기(아이맥/맥북 등)가 같은 Apple ID로 연결되어 있으면,
    'Handoff(연속성)' 기능 때문에 클립보드가 기기끼리 자동 공유됩니다.
    이때 전송 도중 다른 기기에서 복사한 내용이 끼어들면,
    엉뚱한 내용이 전송되거나 전송이 실패할 수 있습니다.

  - 어떻게 끄는가 (전송하는 Mac에서):
    시스템 설정 > 일반 > AirDrop 및 Handoff
    → '이 Mac과 iCloud 기기 간 Handoff 허용' 끄기
    (또는 작업하는 동안만 블루투스를 잠깐 꺼도 됩니다)

▶ 주의사항:
  - 전송 중에는 마우스/키보드를 조작하지 마세요
  - 카카오톡이 로그인되어 있어야 합니다
  - 화면이 잠기지 않도록 해주세요
  - 친구 이름이 카카오톡에 등록된 이름과 정확히 일치해야 합니다

▶ 종료 방법:
  - 터미널 창을 닫거나 Ctrl+C를 누르세요

▶ 필요한 파일:
  - kakao_web.py (메인 프로그램)
  - 카카오톡전송기.command (실행 스크립트)

▶ 문제 해결:
  - "친구를 찾을 수 없습니다" 오류:
    → 카카오톡에서 해당 이름으로 친구가 등록되어 있는지 확인
    → 이름 철자가 정확한지 확인
  
  - "카카오톡 창을 찾을 수 없습니다" 오류:
    → 카카오톡 앱이 실행되어 있는지 확인
    → 카카오톡 창이 화면에 보이는지 확인

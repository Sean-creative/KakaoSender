====================================
  카카오톡 자동 전송기 사용 방법
====================================

▶ 실행 방법:
  1. '카카오톡전송기.command' 파일을 더블클릭하세요.
  2. "확인되지 않은 개발자" 경고가 뜨면:
     - 파일을 우클릭(Control+클릭) > "열기" 선택
     또는
     - 시스템 설정 > 개인정보 보호 및 보안 > "그래도 열기" 클릭
  3. 웹 브라우저가 자동으로 열립니다.

▶ 최초 실행 시:
  - 필요한 패키지가 자동으로 설치됩니다 (1-2분 소요)
  - 인터넷 연결이 필요합니다

▶ 사용 방법:
  1. 웹 브라우저에서 "엑셀 파일을 선택하세요" 클릭
  2. 회원 목록이 담긴 .xlsx 파일 선택
  3. "카카오톡 전송 시작" 버튼 클릭
  4. 카카오톡이 자동으로 열리며 메시지 전송이 시작됩니다

▶ 필터링 조건:
  - 등록형태: 이월, 재등록, 신규
  - 연령: 20대, 30대
  위 조건에 해당하는 회원에게만 메시지가 전송됩니다.

▶ 주의사항:
  - 전송 중에는 마우스/키보드를 조작하지 마세요
  - 카카오톡이 로그인되어 있어야 합니다
  - 화면이 잠기지 않도록 해주세요

▶ 종료 방법:
  - 터미널 창을 닫거나 Ctrl+C를 누르세요

▶ 필요한 파일:
  - kakao_web.py (메인 프로그램)
  - 카카오톡전송기.command (실행 스크립트)

====================================
  카카오톡 자동 전송기 사용 방법
  (v1.0.18 기준)
====================================

▶ 실행 방법:
  1. '카카오톡전송기.command' 파일을 더블클릭하세요.
  2. "확인되지 않은 개발자" 경고가 뜨면:
     - 파일을 우클릭(Control+클릭) > "열기" 선택
     또는
     - 시스템 설정 > 개인정보 보호 및 보안 > "그래도 열기" 클릭
  3. 웹 브라우저가 자동으로 열립니다.

▶ 최초 실행 시:
  - 필요한 패키지가 자동으로 설치됩니다 (3-5분 소요)
  - 인터넷 연결이 필요합니다

▶ 권한 설정 (최초 1회):
  프로그램 실행 시 아래 권한을 허용해주세요:
  
  1. 접근성 권한 (필수):
     시스템 설정 > 개인정보 보호 및 보안 > 손쉬운 사용(접근성)
     → "터미널" 또는 사용 중인 터미널 앱 허용
     ※ v1.0.15부터: 이 권한이 없으면 전송 시작 시 화면에 안내 패널이 뜨고
        전송이 시작되지 않습니다(한 명도 발송되지 않는 사고 방지).
        "권한 요청 팝업 열기" 버튼으로 시스템 설정을 바로 열 수 있으며,
        권한을 켜면 자동으로 전송이 이어집니다. 안 되면 터미널을 완전히
        종료(Cmd+Q)했다가 다시 실행하세요.
     ※ 접근성 권한 하나면 충분합니다. (화면 녹화 등 다른 권한은 필요 없습니다.)

▶ 사용 방법:
  1. 웹 브라우저에서 "엑셀 파일을 선택하세요" 클릭
  2. 회원 목록이 담긴 .xlsx 파일 선택
  3. "카카오톡 전송 시작" 버튼 클릭
  4. 카카오톡이 자동으로 열리며 메시지 전송이 시작됩니다

▶ 엑셀 파일 형식:
  다음 컬럼이 필요합니다:
  - 이름: 카카오톡 친구 이름과 동일해야 함
  - 등록형태: 이월 / 재등록 / 신규
  - 연령: 20대 / 30대 / 40대 등

▶ 필터링 조건:
  - 등록형태: 이월, 재등록, 신규
  - 연령: 20대, 30대
  위 조건에 해당하는 회원에게만 메시지가 전송됩니다.

▶ 친구 검증 기능 (접근성 API):
  - 메시지 전송 전 카카오톡에서 친구 존재 여부를 자동으로 확인합니다
  - 친구가 없으면 해당 대상자는 건너뜁니다
  - 검증 방식: 접근성(AX) API로 친구 이름을 정확한 문자열로 읽어 비교합니다
    (화면 크기·표시 상태와 무관하게 정확 → 긴 이름·이모티콘도 정확히 처리)
  - 검증에 실패하면 잘못 보내지 않고 '실패 목록'에 남깁니다 (오발송 방지)

▶ 이모티콘이 포함된 이름 [중요]:
  - 이름에 이모티콘이 있어도 전송할 수 있습니다.
  - 단, 안전(오발송 방지)을 위해 이모티콘이 포함된 이름은
    "카카오톡 표시 이름과 글자·이모티콘까지 완전히 동일"할 때만 전송됩니다.
    (이모티콘을 뗀 텍스트만 같은 다른 친구에게는 절대 보내지 않습니다)
  - 주의: 이모티콘을 직접 손으로 입력(다른 키보드·이모지 피커)하면,
    화면상 같은 모양이라도 내부 코드(유니코드)가 달라 전송이 안 될 수 있습니다.
    예) 하트가 ❤ 와 ❤️ 처럼 보이지만 서로 다른 문자로 취급될 수 있음
  - 권장: 엑셀에 이모티콘 이름을 넣을 때는
    카카오톡에 표시된 이름을 그대로 "복사 → 붙여넣기" 하세요.
  - 텍스트 없이 이모티콘/기호만으로 된 이름은 지원하지 않습니다.

▶ 모의 전송(Dry-Run) 모드:
  - 실제 메시지를 보내지 않고 전송 과정을 시뮬레이션하는 테스트 기능입니다
  - 웹 화면에서 "모의 전송 (테스트)" 체크박스를 활성화하면 동작합니다
  - 친구 검색 및 검증까지는 동일하게 수행하고,
    채팅방을 1초간 열었다가 닫는 것으로 실제 전송을 대체합니다
  - 새로운 수신자 목록을 처음 등록할 때 사전 검증 용도로 활용할 수 있습니다

▶ 이미지 첨부 (사진 1장):
  - 메시지와 함께 사진 1장을 보낼 수 있습니다.
  - 웹 화면에서 "이미지 첨부"로 사진을 고르면 됩니다 (지원: png/jpg/gif/webp/heic 등, 최대 20MB)
  - 전송 순서 선택: "사진 먼저"(기본) 또는 "텍스트 먼저"
  - 사진은 클립보드로 붙여넣어 전송되며, 텍스트와 함께 각 대상자에게 보냅니다.
  - 전송 중에는 이미지를 바꿀 수 없습니다(섞임 방지). 멈춘 뒤 변경하세요.

▶ 패스트 모드 (빠른 전송):
  - 단계별 대기를 최소화하고(고정 대기 → 상태 폴링), 대상 사이 대기를 없애
    전송 시간을 크게 단축합니다. (예: 50명 기준 약 12분 → 약 4~5분)
  - 웹 화면에서 "패스트 모드" 체크박스를 활성화하면 동작합니다
  - ⚠️ 주의: 대상 사이 대기를 제거하므로 빠른 연속 발송이 카카오톡의
    스팸/매크로 탐지에 걸려 계정이 제한될 수 있습니다. 위험을 감수할 때만,
    가능하면 인원을 나눠서 사용하세요.

▶ 동명이인 처리:
  - 전송 대상에 같은 이름이 2명 이상 있으면 전송이 차단됩니다
  - 엑셀에서 해당 이름을 구분(예: 닉네임 추가)한 후 다시 시도해주세요

▶ 보편적 클립보드(클립보드 공유) 끄기 [중요]:
  - 무엇이 문제인가:
    이 프로그램은 메시지와 이름을 '복사 → 붙여넣기'(클립보드)로 처리합니다.
    여러 애플 기기(아이맥/맥북 등)가 같은 Apple ID로 연결되어 있으면,
    'Handoff(연속성)' 기능 때문에 클립보드가 기기끼리 자동 공유됩니다.
    이때 전송 도중 다른 기기에서 복사한 내용이 끼어들면,
    엉뚱한 내용이 전송되거나 전송이 실패할 수 있습니다.

  - 어떻게 끄는가 (전송하는 Mac에서):
    시스템 설정 > 일반 > AirDrop 및 Handoff
    → '이 Mac과 iCloud 기기 간 Handoff 허용' 끄기
    (또는 작업하는 동안만 블루투스를 잠깐 꺼도 됩니다)

▶ 주의사항:
  - 전송 중에는 마우스/키보드를 조작하지 마세요
  - 카카오톡이 로그인되어 있어야 합니다
  - 화면이 잠기지 않도록 해주세요
  - 친구 이름이 카카오톡에 등록된 이름과 정확히 일치해야 합니다

▶ 종료 방법:
  - 터미널 창을 닫거나 Ctrl+C를 누르세요

▶ 필요한 파일:
  - kakao_web.py (메인 프로그램)
  - 카카오톡전송기.command (실행 스크립트)

▶ 문제 해결:
  - "친구를 찾을 수 없습니다" 오류:
    → 카카오톡에서 해당 이름으로 친구가 등록되어 있는지 확인
    → 이름 철자가 정확한지 확인
  
  - "카카오톡 창을 찾을 수 없습니다" 오류:
    → 카카오톡 앱이 실행되어 있는지 확인
    → 카카오톡 창이 화면에 보이는지 확인
